# Git Practice Project

## Project Overview
This repository is created as part of a DevOps Git practice exercise.

The project demonstrates basic Git workflows including:
- Repository creation
- Committing code
- Pushing changes to GitHub
- Managing project documentation
- Handling production issues using Git commands

The project consists of two components:

- trainee_backend : Backend service
- trainee_frontend : Frontend interface

## Installation

Clone the repository:

git clone git@github.com:harshmanek1001/git_practice.git

Move into the project directory:

cd git_practice

---

## Running the Backend

cd trainee_backend  
npm install  
node server.js

---

## Running the Frontend

Open the following file in a browser:

trainee_frontend/index.html
